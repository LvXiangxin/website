Research experience

Computational Communication Collaboratory 
03/2023 - present

Serving as a research assistant to Associate Professor Cheng-jun Wang, I participate in weekly laboratory reading sessions and group meetings.
I led a project utilizing web scraping techniques to gather historical data of 88,937 individuals from Wikipedia based on the Pantheon dataset. This project involved employing text analysis techniques such as Doc2vec word embedding models and OLS regression models to analyze the influence factors of popularity from the perspectives of moralization and narrative structure. Additionally, causal inference methods were employed for verification. Several papers were produced as outcomes of this project.

Communication Data Mining Competition
 04/2023 - 08/2023

Exploring the diffusion patterns of Weibo hot events using the BERT model. Natural language processing was conducted on 4000 Weibo hot events and the top 1000 comments for each event. The BERT model was utilized for preprocessing, feature extraction, and fine-tuning. This facilitated both text classification tasks and regression tasks for predicting event popularity values. The classification tasks achieved accuracies of over 0.75 on both training and testing sets, while the average absolute error for regression tasks decreased from 0.7 to 0.068.
